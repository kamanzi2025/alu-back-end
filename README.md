# API Project – Employee TODO List

This project contains Python scripts that interact with a REST API to gather employee TODO list data.

## Task 0: Gather data from an API

- The script `0-gather_data_from_an_API.py` fetches the TODO list of a given employee.
- It uses the [JSONPlaceholder API](https://jsonplaceholder.typicode.com/) for fake data.
- It prints how many tasks the employee has completed out of the total.

## Usage

```bash
./0-gather_data_from_an_API.py <employee_id>
